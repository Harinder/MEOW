MEOW is an experimental mobile middleware.

Requires Django 1.2, Python 2.6 

Django install
--------------
Install django via easy_install-2.6

Requires django-jsonrpc - http://github.com/samuraisam/django-json-rpc

Running tests
------------

Go to 'meow' directory, and run 

$ ./manage.py test
